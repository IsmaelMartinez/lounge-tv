# Contributing to Lounge TV

Contributions are welcome. Please open an issue first to discuss what you'd like to change.

## Guidelines

Keep it simple. Lounge TV is a single HTML file by design. Avoid adding build steps, frameworks, or npm dependencies.

Test your changes on both desktop browsers and, if possible, an LG webOS TV (or at least a mobile browser to verify responsive behaviour).

## Pull requests

Fork the repo, create a branch, make your changes, and open a PR. Include a clear description of what you changed and why.
